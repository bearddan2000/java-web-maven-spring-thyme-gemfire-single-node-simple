# java-web-maven-spring-thyme-gemfire-single-node-simple

## Description
A POC for springboot web app with thymeleaf support
connecting to a gemfire single node.

## Tech stack
- java
- maven
  - springboot
  - thymeleaf
  - gemfire
- bootstrap
- jquery
- dataTable

## Docker stack
- gemfire/gemfire
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
[Available](http://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
